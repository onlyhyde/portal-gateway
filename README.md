# Portal Gateway

Production-ready AI Agent & MCP Server hosting platform with end-to-end encryption.

## Features

- **Security & Authentication**: API key authentication with lease-based access control
- **Rate Limiting**: Global and per-lease rate limiting with resource quotas
- **Observability**: Prometheus metrics, structured logging, and Grafana dashboards
- **Reliability**: Circuit breakers, graceful shutdown, and timeout handling
- **AI Agent Support**: SSE/Streaming, webhook retry with DLQ, request/response caching

## Target Use Cases

- MCP (Model Context Protocol) server hosting
- n8n workflow integration
- OpenAI Assistant Function endpoints
- Private RAG agents with E2EE
