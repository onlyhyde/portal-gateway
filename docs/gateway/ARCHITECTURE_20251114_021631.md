#### 3. Proxy Endpoint

```
HTTP Method: ANY
Path: /peer/:lease_id/*path

Headers:
  Authorization: Bearer <api_key>  (required)
  Content-Type: <any>
  Accept: <any>

Example:
GET /peer/my-mcp-server/tools/list
Authorization: Bearer sk_live_abc123...

Response:
  - Proxied from backend service
  - Status code preserved
  - Headers preserved (except hop-by-hop)
  - Body streamed

Error Responses:

401 Unauthorized
{
  "error": "invalid_api_key",
  "message": "API key is invalid or expired"
}

403 Forbidden
{
  "error": "access_denied",
  "message": "API key does not have access to lease 'my-mcp-server'"
}

429 Too Many Requests
{
  "error": "rate_limit_exceeded",
  "message": "Rate limit exceeded. Retry after 30 seconds",
  "retry_after": 30
}

502 Bad Gateway
{
  "error": "backend_unavailable",
  "message": "Cannot connect to backend service"
}

504 Gateway Timeout
{
  "error": "backend_timeout",
  "message": "Backend service did not respond within 60 seconds"
}
```

#### 4. WebSocket Proxy Endpoint

```
WebSocket Upgrade:
GET /ws/:lease_id
Upgrade: websocket
Connection: Upgrade
Authorization: Bearer <api_key>

Response:
HTTP/1.1 101 Switching Protocols
Upgrade: websocket
Connection: Upgrade

Then: Bidirectional WebSocket proxy to backend
```

### Connection Pooling

```go
// Portal Client Manager with connection pooling

type PortalClientManager struct {
    relayURL string
    pools    map[string]*ConnectionPool  // lease_id -> pool
    mu       sync.RWMutex
}

type ConnectionPool struct {
    leaseID     string
    connections chan *PortalConnection
    maxSize     int
    currentSize int
}

func (m *PortalClientManager) GetConnection(leaseID string) (*PortalConnection, error) {
    m.mu.RLock()
    pool, exists := m.pools[leaseID]
    m.mu.RUnlock()

    if !exists {
        pool = m.createPool(leaseID)
    }

    select {
    case conn := <-pool.connections:
        // Reuse existing connection
        if conn.IsHealthy() {
            return conn, nil
        }
        // Connection unhealthy, create new one
        return pool.createNewConnection()
    default:
        // No available connection, create new one
        return pool.createNewConnection()
    }
}

func (m *PortalClientManager) ReleaseConnection(conn *PortalConnection) {
    pool := m.pools[conn.leaseID]

    select {
    case pool.connections <- conn:
        // Returned to pool
    default:
        // Pool full, close connection
        conn.Close()
    }
}
```

**Benefits:**
- Reduce connection overhead (dial time ~10-50ms)
- Reuse E2EE sessions
- Automatic reconnection on failure

### Graceful Shutdown

```go
// Graceful shutdown implementation

func (s *Server) Shutdown(ctx context.Context) error {
    log.Info("shutting down gracefully")

    // 1. Stop accepting new requests
    s.httpServer.SetKeepAlivesEnabled(false)

    // 2. Close Portal connections (new requests)
    s.portalManager.StopAcceptingNew()

    // 3. Wait for in-flight requests (with timeout)
    shutdownCtx, cancel := context.WithTimeout(ctx, 30*time.Second)
    defer cancel()

    done := make(chan error, 1)
    go func() {
        done <- s.httpServer.Shutdown(shutdownCtx)
    }()

    select {
    case err := <-done:
        log.Info("shutdown completed")
        return err
    case <-shutdownCtx.Done():
        log.Warn("shutdown timeout, forcing close")
        return s.httpServer.Close()
    }
}
```

**Shutdown Sequence:**
1. Receive SIGTERM/SIGINT
2. Stop accepting new connections
3. Wait for in-flight requests (max 30s)
4. Close Portal connections
5. Flush metrics and logs
6. Exit
